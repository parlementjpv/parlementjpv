Politique de confidentialité

Le site du Parlement JPV respecte la vie privée des visiteurs. Cette page explique quelles données peuvent être collectées et pourquoi.

1) Données collectées

Nous ne demandons pas d’informations personnelles pour consulter le site. Lorsque vous acceptez les cookies, nous utilisons un outil de mesure d’audience (Google Analytics) afin d’obtenir des statistiques (pages vues, durée de visite, type d’appareil, provenance approximative).

2) Cookies

Les cookies de mesure d’audience ne sont déposés que si vous cliquez sur “Accepter” dans le bandeau cookies.

3) Finalité

La mesure d’audience sert uniquement à comprendre la fréquentation du site et à l’améliorer (contenus les plus consultés, performance mobile, etc.).

4) Durée de conservation

Le choix (accepter/refuser) est mémorisé dans votre navigateur via le stockage local (localStorage). Vous pouvez l’effacer en supprimant les données de navigation du site.

5) Gestion de votre consentement

Pour changer d’avis, vous pouvez effacer les données du site dans votre navigateur, puis recharger la page : le bandeau réapparaîtra.

6) Contact

Pour toute question relative aux données et à la confidentialité, contactez l’équipe via les coordonnées indiquées sur le site.
